## 1. เผ่า (4 เล่นได้ + 1 สูญพันธุ์) — map เข้า 5 เมืองพอดี
| เผ่า | เมืองบ้านเกิด | ธีม/บุคลิก | benchmark vibe |
|---|---|---|---|
| **Human (มนุษย์)** | **NeonUtopia** (เมืองมนุษย์ neon) | สมดุล รอบด้าน เทคโนโลยี | จุดเริ่ม onboarding |
| **Elf (เอลฟ์)** | **Solhaven** (แสงทองนิรันดร์ sanctuary) | แสง/ธรรมชาติ/เวท สง่างาม | cozy/สง่า |
| **Dark Elf (ดาร์คเอลฟ์)** | **Nocturne Alley** (สนธยานิรันดร์) | เงา/ลอบเร้น/เวทมืด ลึกลับ | nightlife/PvP |
| **Alien (มนุษย์ต่างดาว)** | **EternityCity** (เมืองลอยฟ้าเหนือเมฆ) | อารยธรรมก้าวหน้า/จิตพลัง/ข้ามมิติ | flagship/ascended |

## 2. ตำนาน Hellbound — ดาวออร์คที่ล่มสลาย (parable)
**เรื่องเล่า:** Hellbound เคยเป็นดาวของ **ออร์ค** ที่มีอารยธรรมยิ่งใหญ่ แต่ **ผู้นำเกิดความโลภ ต้องการครองอำนาจทั้งดวงดาว** → สงครามแย่งชิงอำนาจไม่สิ้นสุด → จุดชนวนหายนะ (ภูเขาไฟปะทุทั่วดาว/อาวุธทำลายล้าง) → **เผ่าออร์คสูญพันธุ์ทั้งดาว** เหลือเพียง **วิญญาณทรมาน (Shadow Wraiths)** ที่ยังวนเวียน
**แก่น (ทำไมขายได้):** สะท้อนโลกจริง อดีต-ปัจจุบัน — ผู้นำที่โลภ ไม่รู้จักพอ นำหายนะมาสู่ทุกคน · เป็น **คำเตือน** ไม่ใช่แค่ฉากผี
**ผูกกับ horror:** อธิบายทั้งภูเขาไฟ + Wraiths + ความร้าง → horror หุบเขามรณะมี "ความหมาย" (story horror แบบ The Mimic — เล่าผ่าน chapter/lore collectible)
**hook เกมเพลย์:** เผ่าที่ยังอยู่ (มนุษย์/เอลฟ์/ดาร์คเอลฟ์/เอเลี่ยน) เดินทางไป Hellbound เพื่อ **ค้นหาความจริง/เก็บมรดกออร์ค/ไม่ให้ประวัติศาสตร์ซ้ำรอย** → เหตุผลให้ไปลุยหุบเขามรณะ

## 3. ผูกกับระบบ Class/Job ([[../CLASS-JOB-SYSTEM-BLUEPRINT.md]])
**เผ่า = affinity เบา ๆ + บุคลิก/ภาพ ไม่ lock class แข็ง** (คงความยืดหยุ่น + solo-viable + ตามนโยบาย no-P2W: เผ่าไม่ใช่ของซื้อเพื่อพลัง)
| เผ่า | affinity (โบนัสเบา) | เล่นคลาสไหนก็ได้ |
|---|---|---|
| Human | สมดุลทุกสาย (ยืดหยุ่นสุด) | ✅ ทุกคลาส |
| Elf | Mage/Hunter (เวท/ธนู) | ✅ |
| Dark Elf | Assassin/Warlock (ลอบ/เวทมืด) | ✅ |
| Alien | จิตพลัง/เทค — สกิลพิเศษเฉพาะเผ่า | ✅ |
| Orc | (สูญพันธุ์) — อนาคต: ปลดล็อก "Revenant Orc" หรือเป็นศัตรู/บอสใน Hellbound | — |

- ⚠️ **เผ่าต้องเลือกได้ฟรี + balance** (ไม่ขายเผ่าเพื่อพลัง; cosmetic skin เผ่าขายได้ ไม่เพิ่มพลัง) ตาม [[feedback-economy-pvp-policy]]

## 4. การนำเสนอเรื่องเล่า (เพราะ "เรื่องเล่าขายได้")
- แต่ละเมือง = ตำนาน/กวีของเผ่าตัวเอง (lore collectible, NPC เล่าเรื่อง, จารึก)
- DeathValley = chapter เปิดเผยโศกนาฏกรรมออร์คทีละบท (story horror)
- ผูกกับ live-ops: event ตามตำนาน/เทศกาลของแต่ละเผ่า

## 5. ผลต่อ production
- **แต่ละเผ่า = custom mesh body + PBR** (เอลฟ์/ดาร์คเอลฟ์/เอเลี่ยน ทรงต่างกัน) → งานใหญ่ฝั่ง Next-Gen Art ([[project-ai-asset-pipeline]]); ChatGPT Pro เจน concept ต่อเผ่า (ต้องชน [[feedback-graphics-benchmark]])
- โค้ด: ระบบเผ่า (เลือกตอนสร้างตัว, affinity, skill เฉพาะเผ่า) ทำ **หลัง Class/Job** (เพราะ interlock กัน) → Cursor prompt-pack แยก
- ลำดับแนะนำ: Class/Job → Race system → (Orc revenant/enemy เป็น content หลัง)
