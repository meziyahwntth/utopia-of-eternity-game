## Style Bible (ใส่ทุก prompt เสมอ)

```
Style: square game item icon, dark fantasy RPG, transparent/black background,
centered weapon, dramatic rim lighting, vibrant color-coded by tier,
Roblox-compatible flat icon, 1:1 aspect ratio, 512x512
```

**Color Tier Reference:**
| Tier | สี หลัก | สี Glow |
|------|---------|---------|
| Common ★1 | Grey/White | None |
| Rare ★2 | Blue | Blue glow |
| Epic ★★★ | Purple | Purple aura |
| Legendary ★★★★ | Gold/Orange | Gold rim light |
| Mythic ★5 | White+Rainbow | Prism shimmer |

---

## Platform Settings

### Leonardo.ai
- Model: **Leonardo Kino XL** หรือ **RPG 4.0**
- Image Dimensions: **512 × 512**
- Guidance: **7–9**
- Negative Prompt: `realistic photo, human, character, watermark, text, blurry, low quality, 3D render`

### ChatGPT (DALL-E 3)
- ใส่ prompt ตรงๆ + ต่อท้ายว่า `square icon format, game UI asset, 512x512`
- ไม่ต้องใส่ `--ar` parameter

### Midjourney v6
- ต่อท้ายทุก prompt ด้วย `--ar 1:1 --style raw --v 6`
- Negative: `--no text watermark human character`

## Workflow หลังได้ภาพ

1. **Save PNG** ตามชื่อที่กำหนดใน `docs/visual-ref/weapons/`
2. **Upscale** บน [Krea.ai](https://krea.ai) → ลาก drop image → Upscale 4x → Download
3. **Upload to Roblox** → Roblox Studio → Asset Manager → Images → Upload → copy `rbxassetid://XXXXXXX`
4. **แจ้งผม** เพื่อ wire `rbxassetid` เข้า WeaponConfig หรือ Radial Menu icon

---

## Tips สำหรับ Leonardo.ai

- ถ้าภาพมี background สีขาว → ใส่ Negative Prompt: `white background`
- ถ้าต้องการ transparent background → เปิด **Remove Background** หลัง generate
- Generate ทีละ 4 รูป เลือกที่ชอบที่สุด 1 รูปก็พอ
- ถ้าไม่พอใจ → กด **Image to Image** แล้ว adjust รูปที่ใกล้เคียงที่สุด
