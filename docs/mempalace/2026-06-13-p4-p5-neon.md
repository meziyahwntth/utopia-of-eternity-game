## 3. Neon Utopia Place + AirTransport (e4ea48a)

AirTransportService V2 มีอยู่แล้ว (TravelTicketWallet) — wire เพิ่ม:
- `boardTransport()` → ตรวจตั๋ว → TeleportAsync ไป EternityCity (94486544638073)
- DeparturePadTrigger → ProximityPrompt G → ShowAirTransportUI
- SpawnRouter → ผู้เล่นใหม่ → Neon ก่อน (ถ้า NeonUtopia PlaceId > 0)
- starter: 5 AirTickets สำหรับผู้เล่นใหม่บน Neon

**งานค้าง (Praphan ทำ Manual):**
1. สร้าง Roblox Place "Neon Utopia" ใน Dashboard
2. ใส่ Place ID ใน `PlaceSecrets.luau` → `NeonUtopia = XXXXXXXXXXXXXXX`
จนกว่าจะใส่ PlaceId > 0 → SpawnRouter ไม่ teleport (ปลอดภัย)
