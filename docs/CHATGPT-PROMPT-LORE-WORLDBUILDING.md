# [PROMPT เริ่มต้นที่นี่ — คัดลอกทุกอย่างตั้งแต่บรรทัดนี้ลงไป]

คุณคือนักเขียนเรื่องแฟนตาซี RPG ระดับมืออาชีพ เชี่ยวชาญด้านการสร้างโลก (World-building) สำหรับวิดีโอเกม คุณต้องสร้าง **Story Bible** ฉบับเต็มสำหรับเกม Roblox ชื่อ **"Utopia of Eternity"**

## กฎเหล็ก (ต้องทำตามทุกข้อ)

1. **ห้ามลอกเลียน** — ชื่อ บอส มอนสเตอร์ NPC เนื้อเรื่อง ต้องไม่ซ้ำกับ Final Fantasy, Dragon Quest, Lineage, Ragnarok, Genshin Impact, MapleStory, Roblox Blox Fruits, World of Warcraft หรือเกมอื่นใด
2. **ใช้ทั้งสองภาษา** — ทุกชื่อให้มีทั้ง **ภาษาไทย** (ชื่อเรียกในเกม) และ **ภาษาอังกฤษ** (ชื่อ ID ใน code)
3. **Tone** — แฟนตาซีไทยผสมอนาคต (Thai-futurism fantasy) มีกลิ่น steampunk ลอยฟ้า ไม่มืดหม่นเกินไป เหมาะ rating ทุกวัย
4. **เชื่อมกับระบบเกมที่มีอยู่** — ทุกเนื้อเรื่องต้องผูกกับ mechanics ด้านล่างนี้

## บริบทของเกม (สิ่งที่มีอยู่แล้ว)

### โลกและสถานที่
- **Eternity City** — เมืองลอยฟ้าในระดับ 2,000 studs เหนือพื้นดิน เป็นศูนย์กลางของเกม
- **Marina District** — ย่านท่าเรือและตลาดกลางอากาศ คลองน้ำไหลอยู่เหนือก้อนเมฆ
- **Aurora Spire** — หอคอยพลังงานที่สูงที่สุด เป็นแหล่งพลังงานของทั้งเมือง
- **Canal Promenade** — ถนนเดินเล่นริมคลอง มีร้านค้าและชาวเมือง
- **Sky Rail Plaza** — สถานีรถไฟท้องฟ้า จุดเชื่อมเมืองต่าง ๆ
- **Death Valley** — หุบเขาใต้เมือง ดินแดนอันตราย
- **Hellbound Zone** — ดินแดนต้องห้ามชั้นล่างสุด
- **Hyper Space** — มิติระหว่างโลก สำหรับเดินทางข้ามเมือง

### ระบบและ Mechanics
- ผู้เล่นมี **ระดับ (Level) 1-149** แบ่งเป็น 7 ชั้น: ผู้เล่นใหม่ → ผู้ฝึกหัด → ชำนาญการ → ผู้เจนสนาม → ผู้เชี่ยวชาญ → ปรมาจารย์ → วีรบุรุษ
- ระบบ **Clan** (กลุ่ม/เผ่า) ทำสงครามชิงพื้นที่เก็บภาษี
- ระบบ **Pet** (สัตว์เลี้ยง): สไลม์, จิ้งจอกไฟ, มังกรจิ๋ว
- ระบบ **Trading P2P** และ **ตลาดค้าขาย**
- ระบบ **Crafting** — สร้างอาวุธจากวัตถุดิบ: หินชิ้นส่วน (StoneFragment), แร่เหล็ก (IronOre), คริสตัลเวทย์ (MagicCrystal)
- **Sky Treasure** — สมบัติลอยฟ้าปรากฏทุก 2 ชั่วโมง
- มอนสเตอร์ใน game มี 5 ประเภท (NpcId 1-5): หุ่น, ซอมบี้, อัศวินมืด, นักเวทย์, บอส

### อาวุธและไอเทม
- StarSword Tier 1-3 (ดาบดาว)
- LeatherArmor, IronArmor

## สิ่งที่ต้องสร้าง (ทำตามลำดับ ครบทุกข้อ)

---

### ส่วนที่ 1: ตำนานกำเนิดโลก (Creation Myth)

สร้างตำนานกำเนิดโลกของ Utopia of Eternity ความยาว 3-4 ย่อหน้า ตอบคำถามเหล่านี้:
- โลกนี้มาจากไหน? ทำไมเมืองถึงลอยฟ้า?
- มีพลังงานอะไรที่ทำให้ทุกอย่างขับเคลื่อน?
- ภัยพิบัติอะไรเคยเกิดขึ้นและทำลายโลกเก่า?
- ใครเป็นผู้สร้างหรือปกครองโลกนี้ในยุคแรก?

---

### ส่วนที่ 2: ประวัติแต่ละเมือง/พื้นที่ (Location Lore)

สำหรับแต่ละพื้นที่ด้านล่าง ให้เขียน:
- **ชื่อเต็มทางการ** (ภาษาไทย + อังกฤษ)
- **ประวัติความเป็นมา** (2-3 ประโยค)
- **บุคคลสำคัญที่เคยอาศัย** (1 คน)
- **ความลับหรือตำนานที่ชาวเมืองซุบซิบ** (1 ข้อ)

พื้นที่: Eternity City · Marina District · Aurora Spire · Canal Promenade · Sky Rail Plaza · Death Valley · Hellbound Zone

---

### ส่วนที่ 3: บอสและมอนสเตอร์ (Boss & Monster Lore)

สร้างให้ครบ 5 ประเภทมอนสเตอร์ (ต่อยอดจาก NpcId 1-5 ที่มีอยู่) + บอสพิเศษ 2 ตัวเพิ่มเติม

สำหรับแต่ละตัว ให้มี:
1. **ชื่อ** (ภาษาไทย + ภาษาอังกฤษ + ชื่อ ID สำหรับ code เช่น `golem_trainee`)
2. **ที่มา** — มันเกิดมาจากไหน? ถูกสร้างโดยใคร หรือเกิดขึ้นเองจากพลังงานอะไร?
3. **บุคลิก/แรงจูงใจ** (สำหรับบอส) — มันต้องการอะไร? ทำไมถึงต่อสู้กับผู้เล่น?
4. **คำพูดที่มันพูดในการต่อสู้** (1-2 บรรทัด)
5. **drop ที่มันทิ้ง** — อธิบายเชิงเรื่องราว (เช่น "เศษกระดูกเหล็กที่หลุดจากร่างกาย" = IronOre)
6. **จุดอ่อน** ที่ผู้เล่นต้องค้นพบ

**มอนสเตอร์ 5 ประเภทหลัก (ต่อยอดจากของเดิม):**

| NpcId | ชื่อเดิม | ให้สร้างชื่อใหม่พร้อมเรื่องราวเต็ม |
|-------|---------|-----------------------------------|
| 1 | หุ่นฝึกหัด | → บอทหุ่นยนต์รักษาการณ์รุ่นเก่า ถูกทอดทิ้ง |
| 2 | ทหารซอมบี้ | → ทหารที่เสียชีวิตในสงครามเมืองลอยฟ้าครั้งแรก |
| 3 | อัศวินมืด | → ผู้พิทักษ์ที่ถูกคำสาปจาก Aurora Spire |
| 4 | นักเวทย์ผีดิบ | → นักวิทยาศาสตร์ที่ทดลองพลังงานต้องห้ามแล้วกลายร่าง |
| 5 | ราชาหุ่น (BOSS) | → หัวหน้าบอทที่ตื่นขึ้นมาหลัง 100 ปี |

**บอสพิเศษ 2 ตัวเพิ่มเติม** (สำหรับเนื้อเรื่องหลัก):
- บอสที่รักษา Death Valley
- บอสสุดท้ายที่ซ่อนอยู่ใน Hellbound Zone

### ส่วนที่ 4: NPC สำคัญ (Named Characters)

สร้าง NPC ที่มีชีวิตจิตใจ 8 ตัว (ไม่ใช่ศัตรู) สำหรับแต่ละตัว:
1. **ชื่อ** (ภาษาไทย + อังกฤษ + code ID)
2. **อาชีพและที่อยู่** (อยู่ที่ไหนใน Eternity City)
3. **บุคลิกสั้น ๆ** (2-3 ประโยค)
4. **ความลับที่ซ่อนอยู่** (ผู้เล่นค้นพบได้จากเควส)
5. **บทพูดแนะนำตัว** (1 บรรทัด เมื่อผู้เล่นเข้าใกล้ครั้งแรก)

NPC ที่ต้องมี:
- นายกเทศมนตรีของ Eternity City
- พ่อค้าอาวุธ (ขาย StarSword)
- ช่างฝีมือ (สอน Crafting)
- หัวหน้ายาม Marina District
- นักวิทยาศาสตร์ประจำ Aurora Spire
- เด็กที่พ่อหายไปใน Death Valley (เควสหลัก)
- นายหน้ารับจ้าง (Mercenary Board)
- ผู้เฒ่าที่รู้ตำนานดั้งเดิม

### ส่วนที่ 5: เนื้อเรื่องเควส (Quest Stories)

#### 5.1 Main Quest Chain (5 เควสต่อเนื่อง)

สร้างเควสหลัก 5 บท โดยแต่ละบทต้องมี:
- **ชื่อเควส** (ภาษาไทย + อังกฤษ)
- **code ID** (เช่น `main_q1_arrival`)
- **NPC ผู้ให้เควส** (จากส่วนที่ 4)
- **เรื่องราวย่อ** (2-3 ประโยค)
- **objective** (ในรูปแบบที่ใส่ใน game code ได้):
  - `kill_npc` / `visit_zone` / `trade_complete` / `use_emote` / `send_chat` / `equip_pet`
  - ระบุ: objective type, targetCount, npcId (ถ้าเป็น kill)
- **บทสนทนาสั้น** — NPC พูดอะไรตอนให้เควส + ตอนเสร็จเควส (2-3 บรรทัดต่อจุด)
- **รางวัล**: XP amount, Credits amount, หรือ Item (ชื่อ itemId)
- **ตะขอสู่บทถัดไป** — NPC พูดอะไรเพื่อดึงผู้เล่นไปเควสต่อไป

โครงเรื่องหลักควรเป็นการค้นพบความลับว่า **ทำไม Death Valley ถึงเต็มไปด้วยมอนสเตอร์** และ **ใครเป็นคนปล่อยพวกมันออกมา**

#### 5.2 Side Quests (8 เควสเสริม)

สร้างเควสเสริมที่ไม่เกี่ยวกับเนื้อเรื่องหลัก 8 เควส ครอบคลุม objectives ต่าง ๆ:

| เควส | Objective หลัก |
|------|---------------|
| 2 เควส | kill_npc |
| 1 เควส | visit_zone |
| 1 เควส | trade_complete |
| 1 เควส | use_emote |
| 1 เควส | equip_pet |
| 1 เควส | send_chat |
| 1 เควส | ผสม kill_npc + visit_zone |

สำหรับแต่ละ side quest ให้มี: ชื่อ (TH+EN+ID), NPC ผู้ให้, เรื่องย่อ, objective+targetCount, บทสนทนา (ให้เควส+เสร็จ), รางวัล

#### 5.3 Pet Companion Quests (3 เควส)

เควสที่ผูกกับระบบ pet โดยเฉพาะ:
- เควสพบสไลม์ฟ้าครั้งแรก → ให้เลี้ยงเป็นสัตว์เลี้ยง
- เควสช่วยเหลือจิ้งจอกไฟที่บาดเจ็บ
- เควสตามหามังกรจิ๋วที่หายไปจากหอคอย Aurora Spire

## รูปแบบที่ต้องการ (Output Format)

กรุณาตอบโดยแบ่งเป็นส่วน ๆ ตามหัวข้อด้านบน ใช้ Markdown Headers
สำหรับแต่ละ Quest ให้แสดง JSON-like block ด้วยเพื่อให้นำไปใส่ใน code ได้ง่าย เช่น:

```
Quest ID: main_q1_arrival
Objective: visit_zone, targetCount: 1, zone: "marina"
XP: 300, Credits: 0, Item: null
NPC: harbormaster_krit
```

ทำทีละส่วนให้ครบ อย่าข้าม อย่าย่อ เนื้อหาต้องใช้งานได้จริงในเกม

## หลังจากได้ Story Bible แล้ว — ขั้นตอนถัดไป

1. **ตรวจสอบ** ว่าชื่อและ ID ไม่ซ้ำกับที่มีในโปรเจกต์
2. **แปลง Quest** ที่ได้เป็น entries ใน `QuestConfig.luau` (Cursor prompt แยก)
3. **แปลง NPC** เป็น `NpcDialogueConfig.luau` + `NpcSpawnConfig.luau` (Cursor prompt แยก)
4. **แปลง Boss Lore** เป็น `NpcDropConfig` entries ใหม่ (เพิ่ม NpcId 6-7)
5. บันทึก Story Bible ไว้ที่ `docs/STORY-BIBLE.md` ในโปรเจกต์
